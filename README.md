**Perisai Digital untuk Memulihkan Limbah Gagasan Mahasiswa**

AmanSuara.id adalah platform inovatif yang dirancang untuk mengatasi fenomena *Chilling Effect* dan *Self-Censorship* di kalangan mahasiswa. Aplikasi ini berfungsi sebagai "perisai" yang membantu mahasiswa menyampaikan kritik dan aspirasi secara aman, cerdas, dan beretika tanpa rasa takut terjerat pasal karet (UU ITE).

> "Nol Limbah Gagasan, Berani Bersuara dengan Pintar." - Zero Waste Aspiration

## üåü Fitur Utama

1.  **Aspiration Shield (Perisai Aspirasi)**
    *   Alat analisis teks pintar yang memindai draf tulisan mahasiswa.
    *   Mendeteksi potensi pelanggaran (SARA, ujaran kebencian, serangan personal).
    *   Memberikan skor keamanan dan rekomendasi perbaikan diksi.

2.  **Kamus Pasal Karet**
    *   Panduan interaktif mengenai pasal-pasal krusial (Pasal 27 UU ITE, Pasal 207 KUHP, dll).
    *   Dilengkapi dengan tips praktis agar kritik tetap tajam namun lolos dari jerat hukum.

3.  **Visual Guide (Komparasi Kritik)**
    *   Contoh nyata perbandingan antara "Kritik Berkelas vs Hinaan" dan "Aman vs Berisiko".

4.  **Zero Waste Dashboard**
    *   Visualisasi dampak gerakan: menampilkan statistik aspirasi yang berhasil "diselamatkan" dan dipublikasikan.

## üöÄ Cara Menjalankan Project

Ikuti langkah berikut untuk menjalankan aplikasi di komputer lokal Anda:

1.  **Clone repository ini:**
    ```bash
    git clone https://github.com/username/amansuara.git
    cd amansuara
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Jalankan server development:**
    ```bash
    npm run dev
    ```

4.  **Buka browser:**
    Kunjungi [http://localhost:3000](http://localhost:3000) untuk melihat aplikasi.
